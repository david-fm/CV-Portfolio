Moody

Primer proyecto significativo en IA aplicada donde lideré una investigación que culminó en implementación práctica. Repliqué un paper científico entrenando un modelo convolucional para transformar audios en vectores, lo que permitió desarrollar junto a mi equipo un sistema innovador que correlaciona imágenes con audios en una base de datos vectorial.

Mi contribución principal se centró en el desarrollo y optimización del módulo ModelLibrary, componente crítico para el funcionamiento del sistema.

Python Deep Learning Redes Convolucionales Vectorización

Receipt System

Proyecto final de grado galardonado con Matrícula de Honor, enfocado en el desarrollo integral de una aplicación móvil conectada a una API con un modelo NLP tuneado con un dataset propio. Basado en arquitectura Transformer, el sistema procesaba automáticamente tickets de compra a partir de un dataset que recopilé y etiqueté personalmente.

Esta experiencia consolidó mis competencias en entrenamiento de modelos Transformer con PyTorch y estableció una sólida base técnica en el desarrollo de soluciones de IA aplicadas a problemas cotidianos.

Tecnologías utilizadas

PyTorch Transformers Desarrollo móvil OCR API REST

Agente Virtual Ayuntamiento de Pamplona

Implementé un sistema híbrido innovador que integra tecnologías clásicas y modernas en el campo del procesamiento del lenguaje natural. Desarrollé un agente virtual que utiliza modelos de lenguaje avanzados para transformar consultas de usuarios en formatos estructurados, incorporando capacidades de búsqueda web mediante Google y utilizando técnicas de vectorización para optimizar y reordenar los resultados según su relevancia contextual.

Adapté la solución para cumplir con las políticas de seguridad municipales que impedían la implementación de un agente conversacional completo.

Tecnologías utilizadas

NLP LLMs Vectorización Búsqueda inteligente Seguridad de datos

Formación de Ayuntamiento de Madrid

Seleccionado por mi expertise en IA, impartí una formación especializada sobre Inteligencia Artificial Generativa al personal del Ayuntamiento de Madrid. El programa formativo abarcó fundamentos teóricos de modelos de lenguaje, limitaciones técnicas, consideraciones legislativas y casos prácticos de implementación en administraciones públicas.

Complementé la formación teórica con una demostración práctica de un agente de atención ciudadana capaz de responder consultas sobre servicios municipales, ilustrando el potencial transformador de estas tecnologías en el sector público.

Clase Master de Big Data UMU

Invitado como especialista por la Universidad de Murcia, impartí una clase magistral en el Master de Big Data enfocada en aplicaciones prácticas de los datos y la inteligencia artificial en entornos empresariales.

A través de casos de estudio reales y ejemplos prácticos, proporcioné a los estudiantes una visión estratégica sobre cómo implementar soluciones basadas en datos para resolver problemas de negocio concretos, preparándolos para generar impacto inmediato en sus futuras posiciones profesionales.
